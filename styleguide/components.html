Grid

The Unity styleguide is based on Bootstrap. The grid is the foundation for the structure of our content. The grid system allows front-end developers to quickly create responsive, mobile ready websites, that are easily scalable between all screen sizes. You can learn more about the grid by reviewing the Bootstrap Docs.

Colors

Color is an integral part of maintaining the Stony Brook brand. The Unity styleguide provides users with the approved colors to be utilized for any official Stony Brook site. As well, Unity also provides a list of supplementary UI colors, when used wisely, will enhance your design.

Buttons

Buttons come in handy when you would like the end user to complete an action. Buttons can be used as call to actions to other resources, pages, or submitting a form, etc.

Our buttons vary in style and color, and can be used with any Stony Brook related website.
