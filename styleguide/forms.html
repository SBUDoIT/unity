Known Issues

  • There's some inline CSS that might be done in a more boostrappy way.
  • It's only been tested on FireFox.
  • Using <input type='number' /> is ugly.
  • There are no bullet points in this list?
  • The JavaScript, especially the AJAX, is a little off.
  • Comments are lacking.
  • Some of the spacing is off.
  • What are our naming conventions for CSS classes, CSS IDs, JavaScript variables, JavaScript event handlers, etc? I would love to have that standardized.
  • This page is dependent on JavaScript, but I think almost all of our forms are!
  • The toggle button looks the same when it has the focus and when it's toggled on.
  • The ugliest JavaScript in the world to get the scroll spy to work.
  • The primary and secondary contact legends aren't "spied on" correctly because they are vertically equivalent.
  • The scroll spy isn't responsive.
  • There are style tags in the body.
